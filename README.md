# Personal Portfolio Website

This website is a personal portfolio that introduces myself, skills, and interests in creating immersive 3D environments with a focus on 3D lighting. It also includes a separate visualization page that demonstrates basic data visualization and creative SVG graphics using JavaScript.

## Purpose of the Website

The purpose of this website is to:
- Present a brief personal profile and technical skills
- Showcase basic front-end development skills using HTML, CSS, and JavaScript
- Demonstrate SVG-based visualizations created programmatically with JavaScript

## Technologies Used

- Clicking on the name toggles its color
- Hovering over skill items and images creates subtle visual feedback
- Hovering over the SVG light bulb turns the light on and off

## Site Navigation

- **index.html**  
  The main page of the website. It contains personal information, education background, skills, an image preview, and interactive elements. Users can read about my background and see my skills at a glance.  
  There is also a button at the bottom of the page that leads to the visualizations page.

- **visualizations.html**  
  A separate page dedicated to visualizations only. Users can explore:
  - A data visualization created with SVG and JavaScript
  - A creative SVG artwork (a light bulb) that lights up when hovered
